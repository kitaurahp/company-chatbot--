# ドキュメントフォルダ

ここに社内規定のファイル（PDF、Word、Excel）を配置してください。

## 対応ファイル形式

- PDF (.pdf)
- Word (.docx, .doc)
- Excel (.xlsx, .xls)

## 例

```
documents/
├── 就業規則.pdf
├── 旅費規定.docx
├── 経費精算ルール.xlsx
├── リモートワーク規定.pdf
└── 福利厚生ガイド.docx
```

## 注意事項

- このフォルダ内のファイルは `.gitignore` に含まれているため、Gitリポジトリにはアップロードされません
- ファイルを追加・変更した場合は、アプリケーションで「ドキュメントを読み込む」ボタンをクリックして再インデックスしてください
